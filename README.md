# The Craft of Digitization - Ser Bhum Brewery
Repository: Digitization-with-Odoo

# Overview
This repository contains files related to a functional Odoo ERP that transforms the company's current paper-based inventory system, which does not meet compliance standards, into an automated system to improve accuracy and streamline inventory management. 

# Files
Snapshots of the company's ERP system, along with its products, such as downloadable PDF and XLSX files.
